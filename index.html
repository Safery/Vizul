Visualizing Project for World Bank Data, Created by @Safery Under 24 hours.

The website makes a "XMLHttpRequest" to gather data from the World Bank servers. This means, the server is not implying on normal API GET request, such reason the algorithm tries to gather data from a 3rd party domain (World Bank). This means that this website will not work on certain browsers unless the developer mode is turned on. This is not a risk to user but it is a safe caution done by modern browsers to prevent any unwanted site from having securty issues. Please install the following plugin on Chrome to run the site perfectly. The plugin only turns on/off developer mode on Chrome so the site can access API from 3rd party domain (World Bank website).
